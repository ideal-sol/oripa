Place draw animation videos here.

The public gacha draw UI uses `draw_video_url` configured on the winning rank.
If the winning rank has no video, or the result is point back, it uses:

- `default.mp4`

Legacy filename fallbacks are not used by the current UI:

- `ss.mp4`
- `s.mp4`
- `a.mp4`
- `b.mp4`
- `c.mp4`
- `d.mp4`
- `e.mp4`
- `point_back.mp4`

Video paths are public, for example `/draw-videos/default.mp4`.
